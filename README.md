# Optcause

**Optcause** is a curated directory of nonprofit and community organizations in New York City for volunteers, students, and newcomers.

## Run

```bash
npm install
npm run dev
```

Then open the URL shown (e.g. http://localhost:5173).

## What's here

- **Directory** — Main page: filters (category, borough, status, type), Top recommended, full org list, How to use, FAQ, Contact CTA, Disclaimer, Footer.
- **Terms of Use** and **Privacy Policy** — Optcause legal pages.
- **Data** — Static seed data in `src/data/organizations.ts` (no backend).

Docs: see `docs/Optcause-2.md`, `docs/CURATED_NYC_LIST.md`, and `docs/DIRECTORY_IMPLEMENTATION.md`.

## Tech

- React 18 + TypeScript + Vite + Tailwind CSS 4.
